# skyalps-virtual-acars
SkyAlps Virtual ACARS is a professional C#/.NET client for MSFS. It uses FSUIPC to track real-time flight data and a Supabase backend for secure cloud logging.  Key Features:  Automated State Machine (Takeoff/Landing)  Precision Landing Rate capture  Pilot Auth &amp; XP via Supabase  Standalone EXE for regional Alpine ops.
